== Description ==
Quick Audio Player is a powerful and extremely customizable responsive audio player for your website.

That can be installed in any wordpress theme, it runs on all major browsers and mobile devices like iPhone, iPad, IOS, Android, Windows mobile or desktop*.


== Quick Audio Player Features =
* HTML5 Audio Player
* Playlist Supports
* Player Duplicator
* File types: .mp3, .wav, .ogg, etc.
* Player Shortcode
* Live Player Editing
* Player Poster Image
* Multiple Player Skins
* Rich Control Settings
* Player Width Customization
* Player Color Customization


== How To Use ==
First, create a new audio player from `Dashboard > Audio Players > Add New Audio Player`

Then use the `[audio_player]` shortcode to display the audio player in any page. The shortcode required the **id** attribute that is the post id of the player.
You can copy the shortcode from the audio player edit page.
Example: `[audio_player id=184]`
